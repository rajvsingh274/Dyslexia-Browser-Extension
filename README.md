# Dyslexia-Friendly Browser Extension

## Overview
AI-based browser extension that simplifies complex web text using NLP and LLMs.

## Features
- Text simplification using LLaMA3 & Mistral
- Readability scoring (ASL, ASW, PDW)
- Real-time webpage transformation
- Dyslexia-friendly UI

## Tech Stack
- JavaScript, Node.js
- NLP, LLMs (LLaMA3, Mistral)
- Ollama

## How to Run
1. Clone repo
2. Install dependencies
3. Run backend server
4. Load extension in Chrome
